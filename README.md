# Quiz-tradutor-russo-Python
um quiz feito em Python com mais de 200 palavras em russo ensinando a tradução e como se pronuncia a palavra



**Objetivo do Projeto:**

- Proporcionar uma experiência divertida e educativa para expandir o vocabulário em russo.
- Utilizar uma abordagem interativa para o aprendizado de idiomas.
- Desenvolver habilidades linguísticas de forma acessível e envolvente.

**Como Funciona:**
- Ao clicar na aplicação, você receberá uma palavra aleatória em russo.
- Instantaneamente, a tradução e a pronúncia em português serão exibidas.
- Explore novas palavras de forma dinâmica e intuitiva, aprimorando seu conhecimento do idioma russo.
